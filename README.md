## Getting Started

You need to install nodejs before you can start the project.

### Prerequisites

- nodejs (v8.3.0)
- npm

### Install nodejs dependencies

With npm

```bash
$ cd ./path/to/tacpoint/
$ npm install
```

### Startup

- development

```bash
$ cd ./path/to/tacpoint/
$ npm start 
```

- production
```bash
$ cd ./path/to/tacpoint/
$ npm start 
```
